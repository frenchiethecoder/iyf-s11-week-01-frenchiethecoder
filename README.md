## Project Description
This project is a simple website built using HTML and CSS to demonstrate basic structure, styling, and the use of semantic HTML elements.

## Technologies Used
- HTML5
- CSS3

## Features
- Semantic HTML structure (header, main, section, footer)
- Simple and clean layout
- Styled background using CSS
- Responsive meta setup

## How to Run
1. Clone this repository
2. Open `index.html` in your browser

## Lessons Learned
I learned how to structure HTML documents using semantic elements and how to link and apply CSS for styling.

## Challenges Faced
I faced challenges with positioning elements and understanding layout structure, but improved by using semantic tags and basic CSS.
